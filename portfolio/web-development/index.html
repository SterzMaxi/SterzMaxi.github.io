Single-Page-Webanwendung

Im Frontend sorgt Vue zusammen mit Bootstrap für eine moderne, responsive Benutzeroberfläche. Vue kommuniziert dabei über API-Aufrufe mit dem Symfony-Backend. Für die Authentifizierung wird Keycloak genutzt, das sowohl im Frontend als auch im Backend eingebunden ist, um eine sichere Benutzeranmeldung und -verwaltung zu gewährleisten.

JavaScript Frontend mit Vue

Ich habe das JavaScript-Framework Vue verwendet, um die Benutzeroberfläche interaktiv und dynamisch zu gestalten. Vue ermöglicht es, wiederverwendbare Komponenten zu erstellen, die den Datenfluss zwischen dem Frontend und Backend effizient handhaben. Dadurch wird die Entwicklung von reaktionsfähigen und leistungsstarken Single-Page-Anwendungen vereinfacht und die Benutzererfahrung verbessert.

PHP Backend mit Symfony

Ich habe das PHP Framework Symfony verwendet, um serverseitig die APIs zu verwalten, die Datenbank per ORM zu managen und Formulare zu validieren. Darüber hinaus bietet Symfony zahlreiche weitere Funktionen, die die Entwicklung effizient und sicher gestalten.
